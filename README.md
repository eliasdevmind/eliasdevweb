# React App

Uma aplicação base em React pronta para desenvolvimento.

## 📋 Requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn

## 🚀 Como Iniciar

### 1. Instalar dependências

```bash
npm install
```

### 2. Executar o servidor de desenvolvimento

```bash
npm start
```

A aplicação abrirá automaticamente em `http://localhost:3000`

## 📦 Scripts Disponíveis

- `npm start` - Inicia o servidor de desenvolvimento
- `npm build` - Cria a versão otimizada para produção
- `npm test` - Executa os testes
- `npm eject` - Expõe a configuração (não é reversível)

## 🎨 Personalizações

- Edite `src/App.js` para modificar o componente principal
- Estilize com CSS em `src/App.css` ou `src/index.css`
- Adicione novos componentes na pasta `src/`
